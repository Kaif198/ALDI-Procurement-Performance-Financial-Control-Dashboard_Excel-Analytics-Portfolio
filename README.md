# ALDI Procurement Analytics Dashboard

Excel-based procurement dashboard managing **€61.5M spend** with supplier scorecarding, cost savings tracking, and budget analysis.

## 📊 Overview

- **Total Spend:** €61.5M across 130+ transactions
- **Cost Savings:** €6.2M tracked (10.1% rate)
- **Suppliers:** 35+ partners across 7 categories
- **Period:** FY 2024-2025

## 🎯 Features

- **Executive Dashboard** - KPI scorecard with visual charts
- **Spend Analysis** - YoY trending by category/supplier/region
- **Supplier Scorecard** - Weighted performance ratings (Quality 30% | Delivery 25% | Cost 25% | Innovation 20%)
- **Savings Tracker** - €6.2M across 12+ initiatives
- **Budget vs Actual** - Quarterly variance analysis
- **Business Case Calculator** - ROI/NPV modeling

## 🔧 Technical Stack

**Excel Features:**
- Advanced formulas: `XLOOKUP`, `SUMIFS`, `INDEX-MATCH`, `IFS`
- Conditional formatting, data validation, dynamic charts
- Structured tables with named ranges

## 🚀 Usage

1. Download the `.xlsx` file
2. Open in Excel 2016+
3. Start with **Executive Dashboard** sheet
4. Replace sample data in **Data Input** sheet to customize

## 💡 Key Insights

- Top 10 suppliers = 68% of spend → consolidation opportunity
- Payment terms optimization could free €2.8M working capital
- 6 suppliers need performance improvement (<3.0 rating)
